Local testing
=============

Run Spilo locally for testing:

.. code-block::bash

  docker run -p 8080:8080 registry.opensource.zalan.do/acid/spiloprivate-9.6:1.2-p1

Run with mock:

.. code-block::bash

  python3 -m --mock
